# meahyeon
김아현의 자기소개

텍스트의 섬세함으로 소통하고, 데이터의 책임감으로 조직을 움직이는 기획자입니다.

국어국문학 전공을 통해 체득한 논리적 사고력을 바탕으로, 1학년 기획부장에서 시작해 2학년 부회장 및 문과대 대의원 당선에 이르기까지 조직의 핵심 운영진으로서 달려왔습니다.
저의 강점은 단순히 계획에 머물지 않고 '0에서 1'을 만드는 기획의 즐거움을 실질적인 성과로 연결하는 추진력입니다. 
학괴 MT와 학과 테마 행사를 총괄하며, 프로그램의 설계부터 실행까지 모든 단계를 논리적으로 구조화했습니다. 
특히 발생 가능한 변수를 사전에 시나리오별로 치밀하게 관리함으로써, 어떤 돌발 상황에서도 프로젝트의 성공적인 완주를 이끌어내는 위기 관리 역량을 증명했습니다.
저의 책임감은 단순히 자리를 지키는 수동적인 태도를 넘어, 조직의 시스템을 최적화하는 능동적인 동력으로 작용합니다.
학생 자치 기구 운영은 물론, 관리형 독서실 조교 활동 당시에도 기존 업무 방식의 비효율을 개선하여 운영 프로세스를 정교하게 다듬었습니다.
이러한 경험을 통해 관리자의 세심한 시선이 조직 전체의 생산성을 어떻게 바꾸는지 체득할 수 있었습니다. 
저는 어떤 낯선 환경에서도 명확한 목표를 설정하고, 팀원들과 소통하며 실질적인 결과를 만들어낼 준비가 되어 있습니다. 전공의 논리성과 현장에서 다진 실행력을 결합해 귀사의 목표 달성에 기여하는 '완주자'가 되겠습니다.
