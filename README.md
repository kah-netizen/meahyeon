# meahyeon
김아현의 자기소개

텍스트의 섬세함으로 소통하고, 데이터의 책임감으로 조직을 움직이는 기획자입니다.

국어국문학 전공의 논리력을 바탕으로, 1학년 기획부장부터 2학년 부회장 및 문과대 대의원 당선까지 조직의 핵심 운영진으로서 쉼 없이 달려왔습니다. 
대규모 MT와 테마 행사를 기획하며 0에서 1을 만드는 과정의 즐거움을 깨달았고, 발생 가능한 변수를 치밀하게 관리하여 프로젝트의 성공적 완주를 이끌어냈습니다.
학생 자치 기구와 관리형 독서실 조교 활동을 통해 증명된 저의 강력한 책임감은 단순히 자리를 지키는 것을 넘어, 조직의 시스템을 최적화하고 성과를 내는 원동력입니다.
어떤 환경에서든 목표를 명확히 설정하고, 팀원들과 함께 실질적인 결과를 만들어낼 준비가 되어 있습니다.
